# install-gnome.sh

Install the GNOME desktop environment on Raspberry Pi OS Lite using an automated script. This script installs GNOME and the required packages on a clean Raspberry Pi OS Lite system. Go to GUIDE: [Install GNOME on Raspberry Pi OS Lite (Ubuntu-like)](https://forums.raspberrypi.com/viewtopic.php?t=373028#p2233233)

![GNOME on Raspberry Pi](https://wobbo.org/screenshots/20250225__373028_Intro_GUIDE_mini.webp)

```
wget https://wobbo.org/install/2026-03-10/install-gnome.sh
chmod +x install-gnome.sh
./install-gnome.sh
```
